# growbasket-tutorial

### Usage

At first install node js if you have not installed yet.
To install node js go to this website https://nodejs.org/en/download/ and Download node js.
Then follow this below steps.

```sh
$ npm install
```

```sh
$ npm start
# Or run with Nodemon
$ npm run serve

# Visit http://localhost:4000
```
